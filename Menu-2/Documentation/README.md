CodA-Nova is a clean and responsive navigation bar built with **HTML5 and CSS3**.  
It is designed as an **evolving UI project**, where new features, layouts, and technologies will be added progressively over time.

> This repository represents the second navigation template of the **CodA-Nova UI series**.  
> Additional navigation patterns and layouts will be added progressively as the project evolves.

## âœ¨ Current Features
- Responsive navigation bar (mobile, tablet, desktop)
- Sticky header that remains visible while scrolling
- Hover effects on menu links and logo
- Clean and semantic HTML5 structure
- Modern CSS3 styling using Flexbox
- Mobile-friendly layout with media queries
- Lightweight (no JavaScript required at this stage)
- Cross-browser compatible
- Well-structured and commented code
- Reusable navigation menu templates and core UI components  
- Fully documented templates in **Markdown**
